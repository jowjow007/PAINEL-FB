# Comunicados em imagem

Solte aqui os arquivos de imagem (`.jpg`, `.jpeg`, `.png`, `.gif` ou `.webp`) que você quer que apareçam como comunicado em tela cheia no painel da TV.

- O painel busca essa pasta sozinho e mostra cada imagem em rodízio, junto com os avisos de texto (se houver).
- Não precisa avisar ninguém nem mexer em código — é só subir a imagem aqui pelo site do GitHub (botão **Add file → Upload files**) e, em até 10 minutos, ela já entra no rodízio da tela cheia.
- Para tirar um comunicado do ar, é só apagar o arquivo desta pasta.
- Se quiser controlar a ordem de exibição, prefixe o nome do arquivo com números (`01-aviso.jpg`, `02-aviso.jpg`...).
- Prefira imagens no formato horizontal (paisagem), próximas de 16:9, para preencher melhor a tela.
